Here is a list of my projects:

Dog Selector Dog Selector is a little toy I built while at Launch Academy. It is a short quiz that asks the user several questions, using the answers to narrow down a list of dog breeds, finally making a few recommendations.
Specs: Rails, PostgreSQL, HTML, Custom CSS
Worth a Yuk Worth a Yuk is another toy I built, largely to practice some front-end skills. The app retrieves a random joke from a database and displays it for the user, with a button to retrieve another joke. At this time, it is still very much a work in progress, and as such contains only a handful of jokes.
Plans for the future include incorporation of AJAX to allow the user to retrieve new jokes without reloading the page.
Specs: Rails, PostgreSQL, HTML, Custom CSS
